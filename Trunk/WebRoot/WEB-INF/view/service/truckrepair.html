    A: 汽车基础电气培训班
    CURRICULUM BACKGROUND 课程背景
    当前柴油车进入电控时代，而想要修好电控柴油车就必须涉及到电气。在传统的国二柴油发动机上只有启动系统与发电机与电有关，现代的国三、国四柴油发动机都是以电脑（ECU)控制燃油喷射系统的，甚至底盘与车身部分也是由电脑控制。所以对汽车维修人面临着电气部分的技术挑战。根据这种需求我们开设了汽车基础电气培训班。
    COURSE DESCRIPTION 课程介绍
    B: 电控柴油共轨精修培训班
    CURRICULUM BACKGROUND 课程背景
    电控柴油共轨精修培训班所开设的课程主要以博世共轨系统为主，电装、德尔福等系统为辅，进行全面讲解分析共轨系统的工作原理以及故障诊断。我们通过五步教学法因人而宜接近一对一教学，通过东风雷诺DCI11发动机试验台与康明斯发动机试验台进行实际故障设置与排查使学员快速掌握柴油共轨维修技术。
    COURSE DESCRIPTION 课程介绍
    C: 电控柴油喷油器精修培训班
    CURRICULUM BACKGROUND 课程背景
    电控柴油喷油器精修培训班开设的课程为共轨喷油器精修，高压油泵精修及电控发动机故障诊断思路。我们进行一对一教学，使学员快速掌握喷油器精修方法及电控发动机故障分析方法。我们要求学员自带故障喷油器，自己动手维修，使其修复至接近出厂标准时予以毕业。（学员所带喷油器维修配件费用自理）。
    COURSE NECESSITY 课程必要性
    目前很多学员反映，维修喷油器时更换了阀组件，车辆当时修好了可用了一段时间又坏了，或更换阀组件后有的出现发动 机功率不平衡、爆燃、甚至难以启动；还有的冒烟、费油等等。
    什么原因造成这样？其实是更换阀组件后没有调试或调试不当没有按照标准的工艺流程装配造成的。喷油器是全车最精密的元件，元件的配合间隙用微米计算的，如果还按传统模式维修，那么显而易见越修越乱，达不到真正维修标准。车主会认为修后的喷油器用不住或用不了，直接换新的。 大型维修服务企业维修国三、国四车辆时，喷油器一部分更换新总成或送出去维修。如果送出去维修不光会降低利润额甚至会造成一部分客户的流失。
