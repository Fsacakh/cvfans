    BASE PROFILE 基地简介
    国家环保标准不断提升，各地要求逐步淘汰黄标车的大背景下，广大商用车售后维修企业对电喷柴油共轨维修技术提升的强烈需求，上海高起汽车文化传播有限公司与咸宁职业技术学院合作建立卡车之友实训基地，配置各类发动机教具、共轨喷油器检测维修试验台、检测仪等教学设备，达到理论实践相结合，经过培训的学员，快速诊断出电控共轨故障点。
